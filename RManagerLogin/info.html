Login

By Samuel Gresham


Login is an in-browser, modular, universal login system for non-sensitive session data.

Login does not use external code to encrypt or decrypt passwords (it only uses internal JS encryption which isnt very good).

This means that it is very lightweight, but also not suitable for applications where security is required.

Login's accounts are machine-based, making them perfect for applications where only one device will be used by any given person.

Login's modularity means that it can be used throughout several web apps with the same accounts.
